Stack Utilizada

Estrutura: Servidor-Cliente

Na estruturação do projeto foi usado o Framework Django para criação de rotas, interação com o Banco de Dados, Exibição de Dados e interação com o usuário

Estilização

Na Estilização do Projeto Foi utilizado Html com presença acentuada do Framework Django para tornar visual toda a lógica construida e o Bootstrap para estilização de cores, tamanhos, fontes, entre outros.
